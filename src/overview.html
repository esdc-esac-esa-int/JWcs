JWcs is an open source JAVA library under GPLV3 license, used for handling system conversion in astronomy.

JWcs is able to :

  • convert from a Coordinate Reference System (CRS) to another one
  • convert from pixel coordinates to world coordinates through a projection and vice versa
You have three ways to use it :
  • as a JAVA library
  • as two graphical user interfaces to handle CRS and projections
  • as a command line

CRS conversion

A CRS contains two different elements :
  • the coordinate reference frame
  • and the coordinate system.
The coordinate reference frame defines how the CRS is related to the origin (position and the date of the origin - equinox , date of observation ) and the coordinate system describes how the coordinates expressed in the coordinate reference frame (e.g. as cartesian coordinates, spherical coordinates or coordinates of a map projection).

An equinox is an astronomical event in which the plane of Earth's equator passes through the center of the Sun.

An epoch of observation is the moment in time when the coordinates are known to be correct. Often, this will be the date of observation, and is important in cases where coordinates systems move with respect to each other over the course of time

Projection

Conversion of intermediate world coordinates (or projection plane coordinates) to celestial coordinates. Projection plane is given and must be computed from pixels coordinates by applying a linear transformation. The list of supported projection is the following:
  • AIT : Hammer-Aitoff
  • ARC : Zenithal equidistant
  • AIR : Airy projection
  • AZP : Zenithal perspective
  • BON : Bonne’s equal area
  • CAR : Plate carrée
  • CEA : Cylindrical equal area
  • COD : Conic equidistant
  • COE : Conic equal area
  • COO : Conic orthomorphic
  • COP : Conic perspective
  • CYP : Cylindrical perspective
  • MER : Mercator
  • MOL : Mollweide’s
  • PAR : Parabolic
  • PCO : Polyconic
  • SFL : Sanson-Flamsteed
  • SIN : Slant orthographic
  • STG : Stereographic
  • SZP : Slant zenithal perspective
  • TAN : Gnomonic
  • ZEA : Zenithal equal-area
  • ZPN : Zenithal polynomial
